# ADR-001: Prefer Copilot v2 exports from React Core

## Status

Accepted

## Context

The app uses Copilot v2 through `@copilotkit/react-core/v2`. The direct `@copilotkit/react-ui` dependency remains in the frontend manifest even though no source file imports it.

## Decision

Remove `@copilotkit/react-ui` as a direct frontend dependency and keep Copilot UI usage on `@copilotkit/react-core/v2`.

## Consequences

- The dependency manifest matches source imports.
- Maintainers have a clearer supported Copilot integration path.
- If the package is retained transitively by npm, it is no longer owned directly by the frontend workspace.
