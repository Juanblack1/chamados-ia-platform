# ADR 001: Extract Governance Model

## Status

Accepted

## Context

The governance view mixes UI rendering with model construction and helper functions. This makes the biggest frontend file harder to review and test.

## Decision

Move governance model construction and helper functions to `frontend/src/views/agents/governanceModel.ts`.

## Consequences

- `GovernanceView.tsx` becomes focused on rendering.
- Governance calculations become easier to inspect independently.
- No UI or behavior change is intended in this cycle.
