# ADR-001: Keep admin access controls as extracted shared components

## Status

Accepted

## Context

The app shell already imports access-control checkbox components from `views/admin/AccessControls`, but `App.tsx` also kept older local implementations with the same names. Those local functions were not referenced by active JSX.

## Decision

Remove the local implementations and keep the extracted components as the single source of truth for admin permission and group controls.

## Consequences

- Future permission and group-control changes have one clear target.
- `App.tsx` becomes smaller and easier to review.
- Runtime behavior remains unchanged.
