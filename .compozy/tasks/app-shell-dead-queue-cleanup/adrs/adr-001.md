# ADR-001: Remove The Stale Queue Implementation From The App Shell

## Status

Accepted

## Context

`App.tsx` already imports the active queue component from `views/tickets/QueueView.tsx`, but a previous queue implementation remains defined in the app shell. It is unreachable through the current route.

## Decision

Delete the stale local queue implementation and its private helpers from `App.tsx`.

## Consequences

- Positive: Future queue changes have a single owner component.
- Positive: The app shell becomes smaller and less misleading.
- Positive: The risk of patching dead code is reduced.
- Negative: None expected, because the removed code is unreachable and covered by lint/E2E.
