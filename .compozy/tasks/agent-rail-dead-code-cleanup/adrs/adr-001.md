# ADR 001: Remove Unused AgentRail

## Status

Accepted

## Context

`AgentRail` is exported but unused. Keeping inactive UI components makes the ticket module harder to scan and can mislead future work into preserving behavior that no current surface depends on.

## Decision

Delete `AgentRail` and its dedicated support code.

## Consequences

- Ticket workspace panels contain only active UI.
- Future agent rail work should be introduced from an explicit requirement instead of kept as speculative code.
