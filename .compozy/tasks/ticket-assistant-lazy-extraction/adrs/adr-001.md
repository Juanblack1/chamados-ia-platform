# ADR-001: Lazy-load the specialist ticket chat

## Status

Accepted

## Context

The ticket specialist chat is valuable in the detail workspace, but dashboard, queue, governance and admin flows do not need the assistant-ui runtime immediately. Importing assistant-ui from `App.tsx` couples optional chat code to the main app module.

## Decision

Move the ticket chat into a lazy component and keep shared AI text normalization in a small helper module.

## Consequences

- Assistant-ui code is isolated from the app orchestrator.
- Ticket chat remains available in the workspace.
- Future chat-specific changes can be made in a smaller file.
