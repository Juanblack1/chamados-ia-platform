# ADR-001: Use ASCII-safe separators for operational copy

## Status

Accepted

## Context

The frontend source already favors ASCII text. Two SLA strings still render with a corrupted middle-dot artifact, which can be visible in the UI and in review output.

## Decision

Use ` - ` as the separator for compact SLA copy.

## Consequences

- Source text remains stable across encoding contexts.
- The UI avoids visible mojibake in operational labels.
- This does not change layout, data or behavior.
