# ADR-001: Keep App Shell Focused By Deleting Legacy Local Views

## Status

Accepted

## Context

The app shell has grown through feature work and still contains local copies of screens that have already moved to modular views. These local copies are not mounted by the current route logic.

## Decision

Delete the unreachable local intake and detail screen implementations from `App.tsx`, while keeping shared helpers that the active workspace still uses.

## Consequences

- Positive: Future work has fewer false targets.
- Positive: TypeScript and review output become easier to reason about.
- Positive: App shell size decreases without changing behavior.
- Negative: None expected if lint/build/E2E pass.
