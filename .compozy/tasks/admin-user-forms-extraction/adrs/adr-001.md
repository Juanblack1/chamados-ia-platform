# ADR 001: Extract Admin User Forms

## Status

Accepted

## Context

The admin user-management file mixes route-level views with dense form components. The form components have their own API calls, RBAC controls and local state.

## Decision

Move user form/support components into `frontend/src/views/admin/UserForms.tsx`.

## Consequences

- `UserManagementViews.tsx` becomes easier to scan.
- Form behavior remains isolated and easier to modify.
- No user-facing behavior change is intended.
