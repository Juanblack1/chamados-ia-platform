# ADR-001: Reuse only the frontend dev server for local E2E runs

## Status

Accepted

## Context

Local Playwright runs have repeatedly failed before executing tests because Vite remains bound to port `5174`. Reusing the backend would risk test-state leakage because the backend uses in-memory storage during E2E.

## Decision

Allow Playwright to reuse the frontend dev server outside CI, while keeping backend server reuse disabled.

## Consequences

- Local repeated E2E runs are less brittle.
- CI remains strict about occupied ports.
- Backend test state remains isolated between runs.
